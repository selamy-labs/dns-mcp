# Contributing

Follow the [Selamy Labs organization-wide contribution guidelines][org-guidelines]
for workflow, quality, privacy, security, and style requirements.

## Architecture documentation

Architecture diagrams MUST be updated in the same PR that changes the behavior
or spec they describe.

When a guarded source change has no diagram impact, add the
`no-diagram-impact` label and include a
`Diagram impact: none - <reason>` line in the pull request body.

[org-guidelines]: https://github.com/selamy-labs/.github/blob/main/CONTRIBUTING.md
